## P5 Geometric Interpretation

**P5 = (g1−g2) + g1/(g1+g2)** captures the asymmetry between consecutive gaps plus the fractional weight of the first gap. Key properties:

- **P1, P4** project entirely onto positive values (all-positive sequences) — no sign structure
- **P2** projects entirely onto negative values
- **P5 straddles zero** (range −2.54 to +3.84, mean +0.56 for actual zeros)

For GUE gaps (uniform-like spacing): |g1−g2| tends to be smaller → P5 clusters near 0.5 → possibly tighter
For Poisson gaps (exponential, high variance): |g1−g2| can be large → P5 spreads widely

The variance ratio for P5 (Poi/GUE = 3.26×) confirms this intuition but is not stronger than P1/P2/P4.
