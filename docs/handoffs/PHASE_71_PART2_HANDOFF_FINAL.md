## Path 2 — Schwarz Reflection and Quadruple Zero Structure

### Priority: Primary target for Part 2

### Mathematical Content

Two independent symmetries of ζ combine to produce the quadruple zero structure:

**Symmetry 1 — Functional equation** (proved Phase 70 as `riemannZeta_zero_symmetry`):
```
ζ(s) = 0  →  ζ(1−s) = 0
```

**Symmetry 2 — Schwarz reflection** (Path 2 target):
```
ζ(s̄) = conj(ζ(s))  →  ζ(s) = 0  implies  ζ(s̄) = 0
```
This follows from the real Dirichlet series coefficients: for Re(s) > 1,
ζ(s) = Σ n^{−s} has real coefficients, so ζ(s̄) = conj(ζ(s)) by direct
computation. The identity principle extends this to all s ≠ 1.

**Combined — Quadruple structure:**
```
ζ(s₀) = 0  →  {s₀, s̄₀, 1−s₀, 1−s̄₀} are all zeros
```
The quadruple collapses to a pair only when s₀ = 1−s̄₀, i.e., Re(s₀) = 1/2.
RH asserts: all non-trivial zeros are in the collapsed (paired) case.

### Klein Four-Group Structure

The quadruple is generated by the Klein four-group V₄ = ℤ₂ × ℤ₂ acting on ℂ:
- Generator A: s ↦ s̄ (complex conjugation)
- Generator B: s ↦ 1−s (functional equation reflection)
- Combined AB: s ↦ 1−s̄

At Re(s) = 1/2: generators A and B coincide (s̄ = 1−s when Re(s) = 1/2),
collapsing V₄ → ℤ₂. The critical line is the fixed-point set of this collapse.

**Quaternion seed connection (AIEX-435–440):** The V₄ structure maps to
quaternion representation — scalar part Re(s), vector triplet encoding the
two ℤ₂ symmetries plus their combination. Collapse at Re(s) = 1/2 forces pure
imaginary form relative to the 1/2 shift, connecting to Cayley-Dickson
inheritance tower constraints. Record this connection; do not attempt to
formalize it in this phase.

## Proof Route for `riemannZeta_conj`

### Step 1: Prove for Re(s) > 1 via Euler product

Use `riemannZeta_eulerProduct_exp_log`:
```
cexp(∑' p : Primes, −log(1 − p^{−s})) = riemannZeta s    (Re(s) > 1)
```

Conjugate both sides:
```
conj(riemannZeta s)
= conj(cexp(∑' p, −log(1 − p^{−s})))
= cexp(conj(∑' p, −log(1 − p^{−s})))      [exp_conj]
= cexp(∑' p, conj(−log(1 − p^{−s})))      [conj_tsum]
= cexp(∑' p, −log(1 − p^{−conj s}))       [log_conj + cpow_conj]
= riemannZeta (conj s)
```

**Branch cut conditions required (for `log_conj` and `cpow_conj`):**
- `(p : ℂ).arg ≠ π` — holds since p is a positive prime (arg = 0)
- `(1 − p^{−s}).arg ≠ π` — holds for Re(s) > 1 since `|p^{−s}| = p^{−Re(s)} < 1`,
  placing `1 − p^{−s}` in the right half-plane (arg ∈ (−π/2, π/2))

This gives a genuine theorem:
```lean
theorem riemannZeta_conj_of_re_gt_one (s : ℂ) (hs : 1 < s.re) :
    riemannZeta (starRingEnd ℂ s) = starRingEnd ℂ (riemannZeta s)
```

**Anticipated difficulty:** Moderate. The branch conditions require norm estimates
(`Complex.abs_cpow_eq_rpow`, `Real.rpow_lt_one`) that should be available.
If this step is too involved, mark with `sorry` and proceed — the subsequent
steps do not depend on it for the named-axiom fallback.

### Step 2: Extend to all s via identity principle (or introduce axiom)

**Aspirational proof:** Both `s ↦ riemannZeta (conj s)` and `s ↦ conj(riemannZeta s)`
are anti-holomorphic on ℂ \ {1} (composition of analytic ζ with the
anti-linear map conj). They agree on Re(s) > 1. The identity principle for
real-analytic functions gives agreement everywhere. Search for:

| Mathlib lemma | Purpose |
|---|---|
| `AnalyticOn.eq_of_frequently_eq` | Identity principle for analytic functions |
| `Complex.eq_on_open_of_differentiableOn` | May cover the extension step |
| `Filter.Tendsto` arguments | Accumulation point in Re(s) > 1 |

**Note:** Mathlib's identity principle applies to *holomorphic* functions.
Both sides are anti-holomorphic (not holomorphic), so the principle must be
applied after reformulation (e.g., to `s ↦ [riemannZeta (conj s) − conj(riemannZeta s)]`
as a function of `conj s`, which IS holomorphic in `conj s`).

**Cauchy integral formula approach (preferred over bare identity principle — AIEX-456):**

A more concrete Lean path than searching for a standalone identity principle lemma,
inspired by the "missing link" proof in the fractional sums analytic continuation
(Lines That Connect, linesthatconnect.github.io). The technique:

For fixed s₀ ∈ ℂ \ {1}, choose a circle C centered at s₀ small enough to avoid 1.
Both `g₁(s) = riemannZeta (starRingEnd ℂ s)` and `g₂(s) = starRingEnd ℂ (riemannZeta s)`
can be expressed via Cauchy's integral formula on C. Since both equal on Re(s) > 1
(from Step 1) and the integrand converges uniformly on compact sets by
`differentiableAt_riemannZeta`, their derivatives agree — hence they agree everywhere.

In Lean, search for:
```
Complex.tendsto_uniformly_of_differentiableOn
-- or
AnalyticOn.eq_of_frequently_eq
```
The set {s : 1 < s.re} is open and intersects any neighborhood, providing the
accumulation point needed for `eq_of_frequently_eq`.

This is cleaner than a full identity principle argument because it uses only
`CauchyIntegral.lean` (confirmed in Mathlib) plus `differentiableAt_riemannZeta`
(confirmed in Mathlib) — no new analytic function theory required.

**If the extension step is not tractable:** introduce:

## Fractional Sums — New Analytic Continuation Handle (AIEX-455, AIEX-458)

**Source:** Lines That Connect, "Extending the Zeta Function with Fractional Sums"
(linesthatconnect.github.io/blog/proof-of-the-fractional-sum-definition-of-the-zeta-function/)

The fractional sum formula provides a NEW definition of ζ valid for Re(s) > 1−m:

```
ζ(s) = 1/(s−1) · d/dz [Σₖ₌₁ᶻ k^(1−s)] |_{z=0}     (Re(s) > 1−m, any m ∈ ℕ)
```

**Why this matters for Phase 71:** Unlike the Euler product (requires Re(s) > 1),
this formula is valid *inside the critical strip*. By choosing m large, ζ(s) is
defined at any point in the strip via a convergent fractional sum. This is a
structurally distinct approach from anything currently in the sedenion stack.

**Lean tractability:** NOT a near-term target. Fractional sums for complex z,
Gregory-Newton interpolation, and Hermite-Genocchi formula are absent from
Mathlib v4.28.0. Document for future phases.

**Structural parallel to sedenion framework:** The algebraic side (sedenion
forcing) constrains σ from below. The fractional sum definition provides ζ from
above, inside the strip. A future connection between these two representations —
via the identity principle — could be a Route 6 to `riemann_critical_line`.

**Weierstrass M-test parallel:** The fractional sums convergence proof uses
C(n−R)^{−(1+ε)} as the uniform bound via the M-test — the exact same strategy
