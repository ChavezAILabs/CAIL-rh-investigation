## Path 2 — Schwarz Reflection and Quadruple Zero Structure

### Priority: Medium — Most tractable of Paths 2–5

### Mathematical Content

Two independent symmetries of ζ combine to produce the quadruple zero structure:

**Symmetry 1 — Functional equation** (proved Phase 70 as `riemannZeta_zero_symmetry`):
```
ζ(s) = 0  →  ζ(1−s) = 0
```

**Symmetry 2 — Schwarz reflection** (NOT yet in Mathlib — Path 2 target):
```
ζ(s̄) = conj(ζ(s))  →  ζ(s) = 0  implies  ζ(s̄) = 0
```
This follows from the real Dirichlet series coefficients: for Re(s) > 1,
ζ(s) = Σ n^{−s} has real coefficients, so ζ(s̄) = conj(ζ(s)) by direct
computation. The identity principle extends this to all s ≠ 1 by analyticity
(`differentiableAt_riemannZeta`).

**Combined — Quadruple structure:**
```
ζ(s₀) = 0  →  {s₀, s̄₀, 1−s₀, 1−s̄₀} are all zeros
```
The quadruple collapses to a pair only when s₀ = 1−s̄₀, i.e., Re(s₀) = 1/2.
RH asserts: all non-trivial zeros are in the collapsed (paired) case.

### Quaternion / Klein Four-Group Structure

The quadruple is generated by the Klein four-group V₄ = ℤ₂ × ℤ₂ acting on ℂ:
- Generator 1: s ↦ s̄ (complex conjugation)
- Generator 2: s ↦ 1−s (functional equation reflection)
- Combined: s ↦ 1−s̄

At Re(s) = 1/2: generators 1 and 2 coincide (s̄ = 1−s when Re(s) = 1/2),
collapsing V₄ → ℤ₂. This is the critical line as a fixed-point set.

**Quaternion mapping (Furey-inspired, AIEX-435–440):**
- Scalar part: Re(s) — invariant under both ℤ₂ symmetries
- Vector triplet: Im(s), the functional equation shift, the conjugation
- Collapse condition: Re(s) = 1/2 forces the quaternion to pure imaginary
  relative to the 1/2 shift — connecting to representation-theoretic
  constraints in the Cayley-Dickson inheritance tower

This intuition is seed-level (AIEX-437, AIEX-438) — record it but do not
attempt to formalize the quaternion connection in this phase.
